vt-hackaton-backend
=========================

Vector Tiles on-the-flyt generation using Tilesplash
<br/>

Install
-----
```
yarn install
```

Develop
-----
```
gulp serve
```
or
```
yarn run start
```

Build
-----
```
gulp build
```

Run ```gulp serve:prod``` to build and run the app

Test
-----
```
gulp test
```

Prettify
-----
```
gulp prettify
```
